# Welcome to Chirpy! 🦋

This is your new Jekyll website powered by [Chirpy](https://github.com/cotes2020/chirpy-starter).

## 🚀 Quick Start

1. **Edit `_config.yml`** - Set your site title, description, avatar, etc.

2. **Add your first post**
   - Create a new file in `_posts/` named `YYYY-MM-DD-title.md`
   - Example: `2025-03-05-hello-world.md`

3. **Run locally** (optional)
   ```bash
   bundle install
   bundle exec jekyll serve
   ```

4. **Deploy to GitHub**
   - Push to your GitHub repository
   - Go to Settings → Pages → Source: Deploy from a branch
   - Branch: `gh-pages`, Folder: `/ (root)`
   - Wait 1-2 minutes, your site will be live at: `https://zhao922-bot.github.io`

## 📝 Writing Posts

```markdown
---
title: Your Title
date: 2025-03-05 12:00:00 +0800
categories: [category1, category2]
tags: [tag1, tag2]
---

Your content here...
```

## 📋 Configuration

Key settings in `_config.yml`:
- `title` - Site title
- `tagline` - Site description
- `avatar` - Profile photo path
- `timezone` - Set to `Asia/Shanghai`
- `social` - Your social media links
